
Console : Logs information associated with a web page: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript code running in the page context

 	  
Enables you to interact with a web page by executing JavaScript expressions in the context of the page

Console.log() - just an information ..  is a method used to log/write the messages on console

Console.warn() - prints warning you need to watch out for it

console.error() - indicates some serious issue. Need to be corrected..
